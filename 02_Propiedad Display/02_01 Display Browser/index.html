  • Introducción a la propiedad display

La propiedad display controla cómo se visualiza un elemento HTML en el navegador dentro del flujo normal del documento y con los elementos HTML que lo rodean. Esta propiedad es muy importante en la maquetación moderna de hoy.

El valor del navegador

Cada navegador posee un documento de estilos CSS por defecto, se puede notar cuando en la consola de desarrollo se lee al inspeccionar un elemento "user agent stylesheets" o "user agent" en la pestaña "Styles", "Rules", dependiendo del navegador en cuestión. Generalmente es la primera pestaña.

Elementos HTML reemplazados y no reemplazados

Al momento de cómo la propiedad display va a visualizar los elementos HTML tiene en cuenta que etiqueta es, no es lo mismo un párrafo(p), un div que son elementos de origen en bloque que una etiqueta span, strong que son elementos en línea. Pero por otro lado, además de tener en cuenta esto, el comportamiento de los elementos puede variar de acuerdo a unas ciertas variables, el ancho, alto, el lado derecho del contenedor, el contenedor. Si, son muchas variables, y aparte hay 2 categorías que van a tener consecuencias sobre como el navegador los visualiza.

Comenzaremos con el más simple de explicar, los elementos HTML no reemplazados, estos son aquellos elementos HTML donde su contenido está determinado en sí mismo, por ejemplo la etiqueta de párrafo <p>, los encabezados <h*>, un <span>, su contenido se expresa entre la etiqueta de apertura y cierre.

Por otro lado están los elementos HTML reemplazados, estos son una referencia a un recurso externo, una imagen, video, audio, documento, en los elementos de formularios input type="image" son un ejemplo de ello. Una etiqueta <img/> es una referencia a un recurso gráfico que tiene un ancho y alto intrínseco, es decir tienen unas medidas concretas dadas sin la intervención de CSS.

CSS maneja los elementos reemplazados diferente, como cuando calcula sus márgenes o el valor auto a su ancho o alto.
