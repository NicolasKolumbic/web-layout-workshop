El objetivo de este Taller es un repaso por aquellas propiedades y técnicas CSS vinculadas estrictamente a la estructuración de proyectos, a lo que en español denominamos Maquetación Web. Para ello se ahondará en Flexbox, Grid Layout, Web Responsive Design, los diferentes posicionamientos y técnicas comunes para el armado de documentos HTML modernos.

Sobre este material

El taller está pensado para todo aquel que tiene un conocimiento de HTML y medio-avanzado CSS, no voy hacer referencia a propiedades relacionadas con las bases de CSS, el color o texto. Mi objetivo es que al momento de estar frente a la tarea de armar una maqueta cuente con un conocimiento profundo de CSS moderno y un abanico de técnicas para hacer frente con técnicas sencillas, modernas y buenas prácticas.

Este repositorio es complementario al Taller y para fines didácticos contiene solo CSS, HTML y un mínimo de Javascript, para conectar con Codepen por ejemplo.

El taller está organizado en módulos para dividir el contenido por temas, desde los temas que creo que son elementales, luego técnicas modernas como Flex Layout, Grid Layout, los complementos de Position, Float, y Selectores, y por último Response web Design. La Introducción son unos conceptos algo básicos sobre CSS que sirven como base para el resto. También de manera no tan profunda, nos tomaremos tiempo para algunas técnicas anticuadas, ya que algunos proyectos antiguos la utilizan, y mi intención es arrojar luz sobre las viejas técnicas también.

Dentro de cada módulo, cada artículo tiene una serie de secciones enumeradas relacionadas a la propiedad. En la parte superior de cada artículo está lo esencial, y a la manera que navegamos hacia abajo se encuentran los detalles u observaciones.
