Cross axis, Cross size y Align Items.

El eje y tamaño transversal tiene un comportamiento por defecto que está estrechamente relacionado con la propiedad align-items. La propiedad align-items solo actúa sobre el eje transversal, define cómo se alinean los elementos flexibles a lo largo de él dentro de una línea flexible.

En el documento HTML vínculado a está sección se configuraron los altos de las etiquetas html, body al 100% para poder configurar el alto del contenedor flexible al 100%, para que este ocupe toda la pantalla (alto y ancho). Entonces nuestro div tiene las siguientes propiedades: display:flex, height:100% y algunas otras relacionadas al color para ser rápidamente visibles. El resultado es que los elementos flexibles contenidos se estiran por el eje transversal en todo el alto disponible del contenedor flexible. El comportamiento se debe al valor por defecto de align-items que es stretch. Este valor distribuirá los elementos flexibles en todo el alto disponible del contenedor flexible.

Si inspeccionamos un contenedor flexible, en la pestaña de Calculado con el checkbox de Show all marcado, en el caso de Google Chrome y Firefox, se puede ver que el valor por defecto es normal. Este valor equivale a stretch.

La altura de un contenedor flexible.

Entonces sabemos que la propiedad afecta solamente al eje transversal, que por defecto es la altura. Si no definimos un alto para el contenedor flexible, este se acomoda a la altura de los elementos flexibles. Si lo definimos el contenedor flexible se adaptará a ese alto y los elementos flexibles crecerán a lo alto tanto como el contenedor.

altura de elementos flexibles

Los elementos flexibles por defecto se extienden en todo el eje transversal si no se configura alguna medida de altura, si se hace, estos se adaptarán a aquella medida.

Inicio transversal y extensión transversal.

Sumado a los 4 primeros conceptos que comentamos en la sección anterior, el inicio transversal (en ingles cross-start) y la extensión transversal (cross-end) son dos puntos en donde se pueden distribuir los elementos flexibles en el eje transversal (cross axis) (Imagen 1). Además de stretch, align-items tiene una serie de valores para controlar cómo se comportan los elementos flexibles dentro del contenedor.
