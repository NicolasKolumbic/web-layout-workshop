Flexbox en la librería Bootstrap.

Bootstrap es una de las librerías más utilizada en la maquetación HTML y CSS. A partir de la versión 4.0.0 está incluido un módulo de reglas CSS para Flexbox.

Para adaptar la UX/UI a diferentes dispositivos, lo que llamamos comúnmente Web Responsive Design, Bootstrap incluye en los selectores de sus clases CSS las siguientes abreviaturas relacionadas a los diferentes tamaños de dispositivos:

Abreviatura Medida
xs dispositivos menores 576px
sm dispositivos mayores a 576px
md dispositivos mayores a 768px
lg dispositivos mayores a 992px
xl dispositivos mayores a 1200px
xxl dispositivos mayores a 1400px * a partir de Bootstrap 5

La documentación es muy clara con lo que mostraremos algunos ejemplo y el enlace a esta. Bootstrap 4.6
