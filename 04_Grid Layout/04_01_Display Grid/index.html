Introducción

Grid Layout es una especificacion relativamente nueva que brinda una nueva forma de estructurar HTML con CSS. Inicialmente puede recordar a lo que era en el pasado diseñar estructuras con la etiqueta HTML table, tiene una similitud, pero el módulo Grid Layout es más versatil comparandolo con lo primero. Incluso, luego de realizar este material, a titulo personal, Grid Layout es propicio para estructurar un documento HTML pensando en lo macro como las columnas de las diferentes parte de nuestro diseño, utilizar medios de consulta(@media) con Grid es muy poderoso en cuanto a lo sencillo. Flexbox podría hacer las mismas estructuras, pero no con la facilidad de Grid. Coincide con Flexbox que funciona a nivel de contenedor, todo los elemento descendientes de primer nivel se convierten en elemento de grilla.

La pregunta valida que podríamos estar pensando es: ¿Qué diferencia hay entre Flexbox y Grid Layout? La respuesta sería que la primera maneja una dimesión donde el eje principal es horizontal o vertical y tenemos un eje transveral donde podemos gestionar los elementos flexibles, pero no tratamos de la mismas forma los dos ejes. El diseño de grilla es un sistema de 2 dimensiones y esta es la diferencia entre los dos, está es su diferencia principal.

Grid permite re ordenar partes del diseño independientemente del orden del documento HTML. Existen funciones flexibles para definir patrones repetitivos para generar grillas. Puede colocar cuadrículas dentro de cuadrículas o, tablas o contenedores Flexbox.

Grid Layout se define a traves de líneas (Grid Lines), estás representan las divisiones entre las celdas (Grid Cell) y tambien el comienzo y final de una de ellas, son la unidad fundamental y desde implicitamente se generan el resto de componentes. Las vérticales se relacionan con las columnas y horizontales para las filas. Una sucesión de celdas en una columna o fila se conoce como Grid Track o simplemente pista. Una agrupación de celdas en filas y columnas delimitada por 4 Grid Lines forman Grid Area, un área puede ser tan pequeña como una sola celda o tan grande como todas las celdas de la cuadrícula. Los Grid Cell son tambien llamados informalmente items, ya que son parte de una estructura mayor que es el contenedor.

Definir display:grid básico

El valor grid o inline-grid funciona a nivel de contenedor, esto significa que los elementos contenidos por elementos con la propiedad display:grid/grid-inline estarán dentro de un diseño de grilla.

Diferencia entre display:grid y display:inline-grid

Al igual que flex y inline-flex, grid y inline-grid tienen la misma relación, el primero trata al contenedor como un elemento en bloque, con lo que no permite que se acomoden otros elementos a sus lados, el otro permite que se ubiquen otro contedores inline-grid u otros a sus lados, similar a los elementos inline-block.
