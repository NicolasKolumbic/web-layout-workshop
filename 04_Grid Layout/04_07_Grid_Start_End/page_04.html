Esta estructuración de celdas está armada con la palabra clave span, note los texto seguidos de Grid Cell n -.... Es importante aclarar, que si bien no es obligatorio, implementar la propiedad grid-template-columns ayuda a mejorar la estructuración por la altura de las celdas.

Grid Cell 1 - cols span 3
Grid Cell 2 - cols span 2, rows span 4
Grid Cell 3 - rows span 4
Grid Cell 4 - cols span 3

Sí aparte de expandir las celdas, queremos moverla, podemos utilizar la sintaxis anterior junto con span. Estos ejemplos tienen definido la propiedad grid-template-rows con la función repeat y el valor 6, con lo cual la grilla está preparada para 6 filas de 70px. Sí movemos o expedimos sobre filas que no existen, estás se generarán automáticamente si el primer parámetro de repeat es distinto a auto-fit.

Grid Cell 1 - cols span 3
Grid Cell 2 - cols span 2, rows span 4
Grid Cell 3 - rows span 4
Grid Cell 4 - cols span 3
